# Plantillas Typst para la Universidad Nacional de San Agustín (UNSA)

Este repositorio contiene una colección de plantillas y formatos estructurados en **Typst** para informes académicos, laboratorios y presentaciones de diapositivas.

## 📂 Contenido del Repositorio

1. **[generic-slide-fresh](file:///home/alvaro9rqc/1_Pacha/5-varios/templates/typs/generic-slide-fresh)**
   - **Autor:** Hecho por Gemini (Google DeepMind).
   - **Descripción:** Una plantilla modular y altamente personalizable de diapositivas con relación de aspecto 16:9, grillas de puntos desbordables, tarjetas apilables, cronogramas dinámicos e iconos vectoriales SVG dinámicos.

2. **[caratula_unsa_typ](file:///home/alvaro9rqc/1_Pacha/5-varios/templates/typs/caratula_unsa_typ)**
   - **Autor:** Hecho por el usuario (propietario original).
   - **Descripción:** Estructura modular estándar para reportes y trabajos entregables de cursos universitarios de la UNSA.

3. **[laboratorio-abet](file:///home/alvaro9rqc/1_Pacha/5-varios/templates/typs/laboratorio-abet)**
   - **Autor:** Hecho por el usuario (propietario original).
   - **Descripción:** Formato diseñado específicamente para el llenado y presentación de informes de laboratorios alineados a las acreditaciones y rúbricas ABET.

4. **[metropilis-unsa](file:///home/alvaro9rqc/1_Pacha/5-varios/templates/typs/metropilis-unsa)**
   - **Autor:** Hecho por Gemini (Google DeepMind).
   - **Descripción:** Plantilla de diapositivas basada en el tema Metropolis, configurada con colores institucionales y soporte para logos dinámicos.
